# تحلیلگر خرپای دوبعدی با اثرات حرارتی، خطای ساخت و کمانش

[![Python](https://img.shields.io/badge/python-3.7+-blue.svg)](https://python.org)
[![License](https://img.shields.io/badge/license-MIT-green.svg)](LICENSE)
[![Tests](https://img.shields.io/badge/tests-65%20passed-brightgreen.svg)]()
[![Platform](https://img.shields.io/badge/platform-Windows%20%7C%20macOS%20%7C%20Linux-lightgrey.svg)]()

یک ابزار متن‌باز و کامل برای تحلیل استاتیکی خرپاهای دوبعدی با در نظر گرفتن تغییرات دما، خطاهای ساخت و پایداری (کمانش). این برنامه به زبان پایتون نوشته شده و از روش اجزای محدود (FEM) بهره می‌برد. هدف از این پروژه ارائه یک هستهٔ محاسباتی شفاف، قابل فهم و توسعه‌پذیر برای تحلیل سازه‌های خرپایی است که می‌تواند در پروژه‌های مهندسی، تحقیق و توسعه نرم‌افزارهای تخصصی استفاده شود.

## ✨ ویژگی‌های کلیدی

- **تحلیل کامل خرپای ۲بعدی** با روش اجزای محدود (روش سختی)
- **اثرات حرارتی** : تغییر طول ناشی از تغییر دما (سراسری و محلی)
- **خطاهای ساخت** : طول اولیهٔ اضافی یا کمبود عضو (`δL₀`)
- **تحلیل کمانش اویلر** برای اعضای فشاری
- **پشتیبانی از چندین دستگاه واحد** : SI, SI-mm, SI-cm, Imperial
- **خروجی‌های گرافیکی** : هندسهٔ اولیه و تغییرشکل‌یافته، نمودار توزیع انرژی
- **گزارش‌گیری جامع** : CSV, JSON و Markdown
- **اعتبارسنجی خودکار** : تعادل انرژی، تعادل نیروها و قرارداد علامت
- **پوشش تست بالا** : ۶۵ تست خودکار با موفقیت ۱۰۰٪
- **انعطاف در روش حل** : حذف مستقیم (elimination) یا روش پنالتی برای شرایط مرزی

## 📖 نحوه استفاده

### 1. آماده‌سازی فایل ورودی (JSON)

یک فایل با ساختار زیر بسازید:

```json
{
  "units": "SI",
  "temperature_change": 0.0,
  "nodes": [
    {"id": 1, "x": 0.0, "y": 0.0, "is_support": true},
    {"id": 2, "x": 2.0, "y": 0.0, "is_support": true},
    {"id": 3, "x": 1.0, "y": 1.0, "is_support": false}
  ],
  "elements": [
    {
      "id": 1,
      "node_i": 1,
      "node_j": 3,
      "A": 0.01,
      "E": 210e9,
      "alpha": 1.2e-5,
      "delta_T": 50.0,
      "delta_L0": 0.001
    }
  ],
  "loads": {
    "node_forces": [
      {"node_id": 3, "Fx": 1000.0, "Fy": -2000.0}
    ]
  },
  "options": {
    "use_sparse": true,
    "bc_method": "elimination",
    "plot_results": true
  }
}
```

### 2. اجرای تحلیل

```bash
python main.py input.json
```

گزینه‌های بیشتر:

```bash
python main.py input.json --output results --format both --plot
```

### 3. دریافت نتایج

پس از اجرا، فایل‌های خروجی زیر تولید می‌شوند:

- `results_elements.csv` : نیروها، انرژی، وضعیت کمانش
- `results_displacements.csv` : جابجایی گره‌ها
- `results_summary.csv` : خلاصه تحلیل
- `results_report.md` : گزارش قابل خواندن
- `results_geometry.png` : نمودار هندسه
- `results_energy_pie.png` : نمودار انرژی
- `results.json` : خروجی کامل ماشین‌خوان

## 🔧 جزئیات پارامترهای ورودی

### گره‌ها (nodes)

| فیلد | نوع | توضیح |
|------|-----|--------|
| `id` | int | شناسه یکتا |
| `x`, `y` | float | مختصات |
| `is_support` | bool | تکیه‌گاه بودن |

### اعضا (elements)

| فیلد | نوع | توضیح |
|------|-----|--------|
| `id` | int | شناسه عضو |
| `node_i`, `node_j` | int | گره‌های دو سر |
| `A` | float | مساحت مقطع |
| `E` | float | مدول الاستیسیته |
| `alpha` | float | ضریب انبساط حرارتی |
| `delta_T` | float | تغییر دمای محلی (°C). اگر تعریف نشود از `temperature_change` سراسری استفاده می‌شود. اگر هر دو موجود باشند جمع می‌شوند. |
| `delta_L0` | float | خطای ساخت (طول اضافی/کمبود) |
| `I` | float | ممان اینرسی (برای کمانش) |
| `effective_length_factor` | float | ضریب طول مؤثر (پیش‌فرض 1.0) |

### بارها (loads)

- `node_forces` : آرایه‌ای از `node_id`, `Fx`, `Fy`

### گزینه‌های حل (options)

- `use_sparse` : استفاده از ماتریس تنک (برای مدل‌های بزرگ)
- `bc_method` : `"elimination"` یا `"penalty"`
- `plot_results` : رسم نمودارها
- `displacement_scale` : مقیاس تغییرشکل (`"auto"` توصیه می‌شود)

## 📊 نمونه خروجی و تفسیر

### فایل `results_elements.csv`

| ستون | توضیح |
|------|--------|
| `id` | شناسه عضو |
| `L` | طول عضو |
| `delta_L_free` | تغییر طول آزاد (حرارتی + خطای ساخت) |
| `N` | نیروی محوری (+ کشش، - فشار) |
| `U` | انرژی کرنشی |
| `P_cr` | بار بحرانی کمانش |
| `buckling_ratio` | نسبت نیروی فعلی به بار بحرانی |
| `buckling_warning` | هشدار کمانش |

### فایل `results_summary.csv`

شامل: آمار کلی، بیشترین جابجایی، هشدارهای کمانش، تراز انرژی و اعتبارسنجی.

## 🧪 تست‌ها و اعتبارسنجی

برای اجرای تمام ۶۵ تست:

```bash
python -m pytest tests/ -v
```

### اصول اعتبارسنجی به‌کار رفته:

1. **تعادل انرژی** : `U_total = 0.5 * U^T * F`
2. **تعادل نیروهای داخلی و خارجی**
3. **قرارداد علامت** : کشش مثبت، فشار منفی
4. **عدم تکینگی ماتریس سختی**

> نکته: در تحلیل‌های حرارتی خالص (بدون بار خارجی)، خطای تعادل انرژی ممکن است تا ۱۰۰٪ گزارش شود که به‌دلیل مقادیر بسیار کوچک انرژی کل سیستم طبیعی است و به‌معنای خطای محاسباتی نیست.

## 🖼️ خروجی‌های گرافیکی

- **هندسه** (`results_geometry.png`) : سازه اولیه (آبی) و تغییرشکل‌یافته (قرمز)

## 🌐 پشتیبانی از واحدهای مختلف

واحدهای قابل قبول در فیلد `"units"` :

- `"SI"` : متر، نیوتن، پاسکال
- `"SI-mm"` : میلی‌متر، نیوتن، مگاپاسکال
- `"SI-cm"` : سانتی‌متر، نیوتن، پاسکال
- `"Imperial"` : اینچ، پوند، psi
